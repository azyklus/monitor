Crate monitor[−][src]

Expand description

The Monitor is a simple Discord bot that hangs out in the Narwhals server. The bot offers several chat games as well as tools to automatically perform certain administrative tasks in order to better keep the server organized.

You can add the bot to your own server here, or you can build your own by following these instructions.
